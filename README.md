# PvZWP-RE
全称Plants vs. Zombies RE-Doom

## 编译

### dotnet build 命令
打开命令提示符，cd到项目的文件夹，输入dotnet build Lawn.sln

### VS2022
打开VS2022，点击上面的生成，再点生成解决方案

## 安装
把资源包与主程放在同一目录下<br>
资源包可以在[PlantsVsZombies.NET原项目](https://github.com/Mewnojs/PlantsVsZombies.NET)的联系方式里找到
