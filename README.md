# PvZWP-RE
全称Plants vs. Zombies RE-Doom<br>

## 编译

### dotnet build 命令
打开命令提示符，cd到项目的文件夹，输入dotnet build Lawn.sln<br>

### VS2022
打开VS2022，点击上面的生成，再点生成解决方案<br>

## 安装
把资源包与主程放在同一目录下<br>
资源包可以在[PlantsVsZombies.NET原项目](https://github.com/Mewnojs/PlantsVsZombies.NET)的联系方式里找到<br>

## 其它
这个项目只有PCDX版和PCGL版<br>
如果发现小游戏不能解锁，请重打冒险模式<br>

## 注意事项
这是一个早期版本，诸多问题请在Issues里提交<br>
自Alpha1.5.0起，我们不再提供编译好的版本，需要自行编译，源代码里没有资源文件，请按上述方法获取资源文件，当然你也可以在旧的发行版里提取
